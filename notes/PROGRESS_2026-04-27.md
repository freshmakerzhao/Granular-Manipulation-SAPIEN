## 3. 当前可用命令（统一入口）

### 3.1 录制 keyframe

```bash
python run_env.py --entry capture \
  --record-file configs/excavator_s010_ee_keyframes.json \
  --keyframe-t-interval 40 \
  --collision off
```

### 3.2 回放 keyframe（实时，不导出视频）

```bash
python run_env.py --entry scene \
  --mode keyframe \
  --keyframes-json configs/excavator_s010_ee_keyframes.json \
  --cpu --collision on \
  --time-scale 7.0 \
  --replay-apply-mode drive
```

### 3.3 回放并导出图片+视频（headless）

```bash
python run_env.py --entry scene \
  --mode keyframe \
  --keyframes-json configs/excavator_s010_ee_keyframes.json \
  --cpu --collision on \
  --time-scale 7.0 \
  --replay-apply-mode drive \
  --headless-replay \
  --frame-interval-steps 5 \
  --headless-video-fps 15 \
  --headless-output-dir outputs/headless_replay/my_run
```

### 3.4 回放并导出建图数据（HDF5）

```bash
python run_env.py --entry collect excavator_s010 \
  --keyframes-json configs/excavator_s010_ee_keyframes.json \
  --cpu --collision on \
  --replay-apply-mode drive \
  --sample-interval-steps 10 \
  --output outputs/mapping_rollouts/my_run_hdf5
```

输出文件：
- `outputs/mapping_rollouts/my_run_hdf5/rollout_data.hdf5`
- `outputs/mapping_rollouts/my_run_hdf5/rollout_data_meta.json`

### 3.5 检查导出数据

```bash
python run_env.py --entry inspect \
  outputs/mapping_rollouts/my_run_hdf5/rollout_data.hdf5
```
